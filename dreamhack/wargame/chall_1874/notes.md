# Dreamhack chall_1874

## 1. 문제 요약
10개의 아이템 중 flag의 인덱스를 50번 연속 맞추는 문제

## 2. 핵심 개념
- pwntools 자동화
- 출력 파싱
- 반복 입력 처리

## 3. 문제 분석
매 라운드마다 아이템 순서가 랜덤으로 섞인다.
출력된 목록에서 flag의 위치를 찾아 그 인덱스를 입력하면 된다.
총 50라운드이며 10초 제한이 있다.

## 4. 풀이 전략
각 줄을 읽으면서 ". " 기준으로 인덱스와 아이템 이름을 분리한다.
아이템 이름이 flag인 줄의 인덱스를 저장한 뒤 입력으로 전송한다.

## 5. 코드 설명
10줄의 아이템 목록을 읽고, flag가 있는 줄의 번호를 찾는다.
찾은 번호를 sendline으로 전송한다.

## 6. 배운 점
문자열을 정규식 없이 split으로도 충분히 파싱할 수 있다.
문제 출력 형식을 정확히 읽는 것이 중요하다.
