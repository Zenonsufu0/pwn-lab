# Dreamhack chall_1114

## 1. 문제 요약
무작위로 생성되는 두 수의 덧셈을 1초 안에 50번 맞춰야 하는 문제

## 2. 핵심 개념
- pwntools를 이용한 자동화
- recv / send를 통한 입출력 처리
- 문자열 파싱

## 3. 문제 분석
프로그램은 다음과 같은 형태로 문제를 출력한다:
1234+5678=?

- rand()로 생성된 두 숫자를 출력
- scanf로 입력을 받아 정답 확인
- 50번 반복
- 틀리면 종료
- 모두 맞추면 flag 출력

→ 사람이 직접 풀기에는 시간 제한 때문에 불가능

## 4. 풀이 전략
- 한 줄 전체를 recvline으로 받는다
- "+"와 "="를 기준으로 숫자를 분리한다
- 두 숫자를 더해서 sendline으로 전송한다
- 이를 50번 반복한다

## 5. 코드 설명

```python
line = p.recvline().decode().strip()
-> 한줄 입력 받기
a, b = line.split("+")
b, _ = b.split("=")
-> 숫자 분리
p.sendline(str(int(a) + int(b)).encode())
-> 계산 후 전송

## 6. 배운점
- pwntools에서 recv/send 흐름 이해
- 문자열 파싱 방법(split)
- 자동화가 필요한 문제 유형
- 1초 제한이 있는 문제는 반드시 스크립트 사용
